# Meetings Implementation Initiative

This folder contains all plans specific to building and refining the Meetings module in Alleato OS.

## Overview

The Meetings module integrates with Fireflies.ai to automatically capture meeting transcripts, extract insights, and track action items. This initiative documents the implementation strategy for meeting management and AI-powered features.

## Planned Features

### Meeting List & Detail Pages
- Meeting list with filtering and search
- Meeting detail page with transcript
- Participant tracking
- Meeting metadata (date, duration, project)

### Fireflies.ai Integration
- Automatic transcript import
- Audio/video file linking
- Meeting synchronization
- Webhook handling for real-time updates

### Meeting Transcripts & AI Insights
- Full transcript display with speaker labels
- AI-generated summaries
- Key bullet points extraction
- Overview and highlights

### Action Item Extraction
- Automatic action item detection
- Task assignment from meetings
- Action item tracking and completion
- Reminders and notifications

## Database Schema

**IMPORTANT**: Meeting data is stored in the `document_metadata` table, NOT a dedicated meetings table.

### Key Tables

- **document_metadata** (filter by `type='meeting'`)
  - Meeting metadata, transcripts, summaries
- **meeting_segments**
  - Chunked meeting content for RAG/AI
  - Vector embeddings for semantic search

### Type Mapping

```typescript
// Import from @/types
import { Meeting } from '@/types'

// Meeting is an alias for document_metadata Row type
```
