## How to Use This Index

### Finding Documentation

1. **Know what you're looking for?**
   - Use the category tables above to find it

2. **Not sure where something is?**
   - Check [DOCUMENTATION-QUICK-REFERENCE.md](./docs/DOCUMENTATION-QUICK-REFERENCE.md) for lookup tables

3. **Need to create new documentation?**
   - Read [DOCUMENTATION-QUICK-REFERENCE.md](./docs/DOCUMENTATION-QUICK-REFERENCE.md) first

   - Use 7-day rule for cleanup
   - Consolidate duplicates
