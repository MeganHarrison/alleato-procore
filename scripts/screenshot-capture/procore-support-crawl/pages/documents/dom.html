La herramienta Documentos es donde los usuarios especificados pueden almacenar y acceder a los archivos importantes de cualquier tipo.

  • Configure una estructura de carpetas centralizada creando carpetas y subcarpetas
  • Establecer y administrar permisos en carpetas y documentos
  • Cree carpetas visibles públicamente para que los licitantes invitados puedan ver los documentos relevantes sin tener que iniciar sesión en Procore (debe tener herramienta de licitación)
  • Cargue o descargue archivos y vea el progreso
  • Consulte y bloquee un documento
  • Administre revisiones y distribuya actualizaciones a usuarios especificados
  • Cree responsabilidad mediante el seguimiento de cuándo y quién descargó cada archivo

Cambios recientes

Extraer puntos de vista de Autodesk® Navisworks® para publicar vistas (19/08/2025)

Procore ha introducido una nueva función que permite a los usuarios agregar puntos de vista desde un archivo de modelo en Autodesk® Navisworks® directamente desde la herramienta Documentos. La función "Agregar vistas" permite a los usuarios seleccionar y publicar hasta 200 vistas, lo que mejora la capacidad de visualizar e interactuar con los datos del modelo. Para obtener más información, consulte Publicar un modelo con la herramienta Documentos.

Experiencia modernizada (19/08/2025)

Procore ha mejorado la herramienta Documentos modernizando su interfaz de usuario (UI) para alinearla con los patrones de diseño actualizados. Las páginas Configuración y Correo electrónico se han reemplazado por páginas de nuevo diseño, que ofrecen una apariencia más moderna sin pérdida de funcionalidad.

Obsolescencia del campo "name_with_path" en la API de documentos (19/08/2025)

Procore ha dejado de utilizar el campo "name_with_path" en la API de documentos, lo que afecta a la API v1.0 para la herramienta Documentos a nivel proyecto. La API v1.0 ha sido reemplazada por la API v2.0, que ofrece un nuevo formato de respuesta y elimina el campo "name_with_path" de las vistas de respuesta.

Renderizado de PDF móvil modernizado (06/27/2025)

Procore ha mejorado la representación móvil de PDF para la herramienta Documentos para iOS y Android. Apryse Viewer ahora proporciona una experiencia de visualización moderna, preparando la herramienta para futuras funciones de marcado.

Procesamiento automático de archivos de modelos 3D para clientes Procore BIM (02/20/2024)

Los archivos de modelo 3D en la herramienta Documentos ahora se procesan automáticamente en procore cuentas que tienen el producto BIM. Consulte View y Descargar modelos 3D en la herramienta Documentos.

Soporte para archivos de modelosd DWG (12/12/2023) 

Procore ahora admite la habilidad para cargar, procesar y ver archivos 3D DWG en la herramienta Documentos. Consulte Ver y descargar modelos 3D en la herramienta Documentos.

Soporte para archivos de modelos IFC (10/04/2023) 

Procore ha agregado soporte para ver y descargar archivos de modelo de Industry Foundation Class (.ifc) en la herramienta Documentos. Consulte Ver y descargar modelos 3D en la herramienta Documentos.

Exportar datos de OBject desde un modelo (13/03/2023) 

Procore ha agregado una función que le permite exportar datos de objetos desde un archivo de modelo en la herramienta Documentos del proyecto. Consulte Exportar datos de objetos desde un archivo de modelo. Una vez completada la exportación, recibirá un correo electrónico con un archivo CSV que contiene los datos.
