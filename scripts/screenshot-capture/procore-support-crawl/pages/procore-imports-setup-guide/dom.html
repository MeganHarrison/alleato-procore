 Importante
Para importar ítems de segmento a la estructura de desglose del trabajo (WBS) de Procore, debe utilizar la aplicación Procore Imports versión 1.4.0 (o posterior). Si está ejecutando la versión 1.3.9 (o anterior), debe desinstalar la aplicación Procore Imports y luego instalar la versión más reciente.

Objetivo

Descargar e instalar Procore Imports en una PC o computadora portátil con una versión compatible de Windows. 

Aspectos para tener en cuenta

  • Procore Imports debe instalarse en un sistema que ejecute Microsoft Windows 10 o posterior. NO es compatible con versiones anteriores de Microsoft Windows, dispositivos iOS o MacOS.

Pasos

  1. Haga clic en el siguiente enlace para descargar la aplicación Procore Imports:

  2. Una vez que la descarga se completa, el instalador se ejecuta automáticamente. Si no se ejecuta, puede:
    • Haga clic en el archivo llamado ProcoreImportsSetup.exe. Puede encontrar este archivo en la ubicación de descarga.
  3. Una vez instalada, Procore Imports se inicia automáticamente. Si no se inicia, haga doble clic en el icono Procore Imports en el escritorio de su computadora. 

    Ahora puede iniciar sesión en su cuenta y comenzar a usar Procore Imports. Consulte Iniciar sesión en Procore Imports.
