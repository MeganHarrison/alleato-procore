## ðŸ”§ Troubleshooting

### "Login failed"
- Verify credentials are correct
- Check if 2FA is enabled (manual intervention required)
- Ensure account has directory access

### "No data captured"
- Verify project ID and company ID
- Check user has permission to view directory
- Try increasing timeout values

### "Screenshots are blank"
- Run in non-headless mode (already default)
- Check if pages require specific viewport size
- Verify network connectivity
