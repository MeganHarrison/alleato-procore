## About This Sitemap

This sitemap is dynamically generated using Nextra's `useConfig` hook, which provides access to the complete page map. The component automatically:

- Lists all pages and folders in the documentation
- Organizes them hierarchically
- Provides clickable links to each page
- Updates automatically when you add or remove pages

To use this component in your own MDX files, simply import and use it:

```tsx
import { Sitemap } from '../components/Sitemap'

<Sitemap />
```
